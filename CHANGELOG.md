# Changelog
All notable changes to the WooCommerce Estimated Delivery Date plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-03-20

### Added
- Initial plugin release
- Gutenberg block for displaying estimated delivery dates
- Admin settings page with following features:
  - Default lead time configuration
  - Store closed days selection
  - Postage closed days selection
  - Display format options
  - Date format selection
- Product-specific lead time settings in product shipping tab
- Shipping method transit times configuration
- Cart and checkout display integration
- Full block customization options:
  - Display type (text/date only)
  - Text alignment
  - Font size, family, and weight
  - Text and background colors
  - Margin settings
  - Border styling
  - Icon selection and positioning
- RTL language support
- HPOS (High-Performance Order Storage) compatibility
- WordPress coding standards compatibility
- Full internationalization support
- Comprehensive documentation

### Technical Implementation
- Object-oriented PHP structure
- WordPress hooks and filters for extensibility
- WooCommerce integration points
- Modern JavaScript with ES6+ features
- Webpack build process
- NPM package management
